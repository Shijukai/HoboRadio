注意
グローバル版及びUI版は開発中のベータ版となります。
ある程度動作確認はしていますが、不具合が起こる可能性があります。あらかじめご了承の上ご利用ください。

導入方法
Assets/Shijukai/Radio/にあるRadio_〇〇.prefabをhierarchyにD&Dしてください

UI版導入時の注意
LilttonとAudioLinkが必要です。事前に導入を済ませておいてください。
また、スペクトラム部分に下記のシェーダーを使用しています。
RiversideTrap 様
【無料シェーダー】AudioLink Spectrum【VRChat想定】
https://booth.pm/ja/items/3051162

グローバル版導入時の注意
ワールドに複数配置すると不具合が起こる可能性がありますので、グローバル版を使用する場合は一つだけの配置に留めてください。

電源の初期状態の設定
導入後、Hierarchy上でRadio_〇〇/Radio_Rootを選択してください。
Inspectorを確認していただくと項目が並んでいると思いますので、
その中のRadio Power On のチェックボックスにチェックが入っていると初期状態で電源がオンになります。
このチェックを外すと初期状態で電源がオフになりますので、お好みで設定ください。
また、不具合の原因となりますのでこれ以外の箇所にはお手を触れないようにお願い致します。

初期チャンネルの設定
導入後、Hierarchy上でRadio_〇〇/Radio_Rootをクリック、
CurrentChannelIndexの数値を0-2の間で変更すると、
0でCH1、1でCH2、2でCH3が初期チャンネルになります。 

電源を入れた時とチャンネル切り替え時のノイズの大きさの調整
Hierarchy上でRadio_〇〇/Radio_Root/Audio Source/ChannelNoiseSEを選択し、
コンポーネント「AudioSource」のVolumeを調整していただくと、音量を下げることができます。
また、ノイズがない方が良い場合はここを0％にして対応をお願いします。
オブジェクトやコンポーネントの削除を行ってしまうと不具合の原因となります。

カラーリングの変更/限定モデルの導入
1. Assets/Shijukai/Radio/にあるRadio_GlobalかRadio_LocalのprefabをhierarchyにD&Dしてください
2. hierarchy上のRadio_〇〇/Radio_Rootの中にあるBlackを削除してください。
3. Assets/Shijukai/Radio/ColorOptionsにあるprefabか、限定モデルのprefabをRadio_〇〇/Radio_RootにD&Dしてください。
4. hierarchy上でRadio_Rootをクリック、
    inspectorのRadioAnimatorの欄に3.で配置したオブジェクトをhierarchyからD&Dしてください。
5. Radio_〇〇/Slider_Colliderをクリック、
    inspectorのKnobの欄にRadio_〇〇/Radio_Root/Radio_Low/Armature/Radio_Root/SliderをD&Dしてください。

UI版の背景について
Assets/Shijukai/Radio/Material/UI/Shijukai_UI_Backgroundのマテリアルを編集することでカラー等の変更ができます。
また、RenderingModeをCutoutに変更して、メインカラーのAlpha（不透明度）を0にすることで透明にすることも可能です。

パブリック化にあたって
VRChatにはパブリックインスタンスではYoutubeなどのVRChat側が信頼しているものを除き、
あらかじめ登録されたドメイン以外を弾く仕様があり、そのままだとパブリックインスタンスでのみラジオが流れません。
パブリック化する際にはウェブサイト上からワールド情報の編集画面を開き、
VideoPlayerAllowedDomainsの欄に「shijukairadio.com」「media.shijukairadio.com」を追加登録してください。
